# Welcome to whatsapp-bot 👋

> This project The bot searches and sends the location of restaurants and fast foods within a 5km radius from the location provided by the user.

## Install

```sh
yarn install
```

## Usage

```sh
yarn dev || yarn build && yarn start
```
